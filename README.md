# ✈️ AéroFormulaires

Portail officiel des formulaires aéroportuaires — 100% HTML/CSS/JS, sans serveur.

## 🚀 Déploiement sur GitHub Pages

1. Créez un dépôt GitHub (ex: `aeroformulaires`)
2. Uploadez le fichier `index.html`
3. Allez dans **Settings → Pages → Branch: main → Save**
4. Votre site sera disponible sur : `https://votre-nom.github.io/aeroformulaires/`

## 📋 Utilisation

- **Page Formulaires** : Recherche, filtre et ouverture des PDF
- **Page Administration** : Ajout et suppression des formulaires
- Les PDF sont stockés localement dans le navigateur (localStorage)
- Impression directe depuis le navigateur
